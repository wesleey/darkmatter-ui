# Welcome to Darkmatter UI

Darkmatter UI is a simple demo design system developed using [Stitches](https://stitches.dev/).

## Components

- [x] Avatar
- [x] Box
- [x] Button
- [x] Heading
- [x] MultiStep
- [x] Text
- [x] TextArea
- [x] TextField

## Installation

npm:

```bash
npm i @darkmatter-ui/react
```

yarn:

```bash
yarn add @darkmatter-ui/react
```
